# Finova Data Viewer – Architecture Report

## Client–Server Model
A client is the browser used by the user. A server processes requests sent by the client.
When the user clicks Search, the browser sends a request to the server, which fetches data and sends it back.

## Full Stack Overview
Frontend: HTML, CSS  
Backend: Handles logic  
Database: Stores data  
APIs: Fetch external data

## Diagram
Client → Server → Database → Server → Client
